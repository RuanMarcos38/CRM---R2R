Como funciona a integração N8N + CRM:
O N8N recebe eventos do CRM via Webhook (novo lead, mensagem, etapa mudou) e executa automações: enviar e-mails, criar tarefas, notificar equipe no WhatsApp, integrar com Google Sheets, etc.

🔧 Como usar no N8N
No seu workflow N8N, adicione um node HTTP Request e configure:
Method: POST
URL: https://seu-projeto.supabase.co/rest/v1/leads
Headers:
  apikey: SUA_SUPABASE_PUBLISHABLE_KEY
  Authorization: Bearer sua-api-key
  Content-Type: application/json
  Prefer: return=representation
O CRM usa o Supabase como backend. Todos os endpoints abaixo são acessíveis diretamente via REST:
BASE URL
https://seu-projeto.supabase.co/rest/v1
GET /leads?empresa_id=eq.{id}&order=created_at.desc Listar leads
POST /leads Criar lead
PATCH /leads?id=eq.{lead_id} Atualizar lead
GET /clientes?empresa_id=eq.{id} Listar clientes
POST /mensagens Enviar mensagem
GET /campanhas?empresa_id=eq.{id} Listar campanhas
GET /conversas?empresa_id=eq.{id} Listar conversas
POST /tarefas Criar tarefa
Headers obrigatórios:
apikey: SUA_SUPABASE_PUBLISHABLE_KEY
Authorization: Bearer [sua-api-key]
Content-Type: application/json
